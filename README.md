# Salmon — Know Your Credit Score

A static single-page credit score quiz app. Deploy to Vercel with zero config.

## Deploy to Vercel

### Option 1: Vercel CLI

1. Install the CLI (if needed):
   ```bash
   npm i -g vercel
   ```
2. From this directory, run:
   ```bash
   vercel
   ```
3. Follow the prompts (log in, link or create a project). Your app will be live at a `*.vercel.app` URL.

### Option 2: Git + Vercel Dashboard

1. Push this repo to GitHub (or GitLab/Bitbucket).
2. Go to [vercel.com](https://vercel.com) → **Add New Project**.
3. Import your repository. Leave **Build Command** and **Output Directory** empty (root is used as static output).
4. Click **Deploy**. Vercel will serve `index.html` at the root.

### Option 3: Drag & Drop

1. Zip this folder (must include `index.html` and `vercel.json`).
2. Go to [vercel.com/new](https://vercel.com/new) and drag the zip into the deploy area.
